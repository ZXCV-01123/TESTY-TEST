Otwieranie i obsluga php my admina:

Zeby otworzyc panel admina bez danych
do paska adresu wpisujemy:
localhost/phpmyadmin

Logowanie do serwera MySQL:

Po otwarciu wiersza polecen pakietu xampp,
przy pomocy przycisku shell, wpisujemy polecenie
mysql -u root -p

mysql - to nazwa serwera DB
-u root - to nazwa uzytkownia z wpisanym uzytkownikiem "root"
-p - to oczekiwanie na nazwe bazy danych

Nastepnie potwierdzamy zapytanie o haslo,
i jestesmy zalogowani.

Po zalogowaniu wszystkie polecenia z wyjatkiem
tych zaczynajacych sie na "\" i polecenia "quit"
konczymy srednikiem.

Skladnia Jezyka SQL:

Polecenie SQL sa podzielone na trzy (3) grupy, ze wzgledu na ich przeznaczenie.

  • DDL - Data Definition Language, definiowanie danych sluzy do tworzenia, modyfikowania i usuwania obiektow
  • DML - Data Manipulation Language, manipulowanie danymi czyli wyszukiwanie i modyfikowanie ich.
  • DCL - Data Control Language, kontrola dostepu, nadawanie i odbieranie uprawnien.

Instrukcje jezyka SQL

Instrukcje SQL skladaja sie z:
slow kluczowych, klauzul, wyrazeni warunkow.

- Wyrazenie okresla co nalezy wykonac,
- Klauzula okresla ograniczenia
- Warunek definiuje ograniczenia klauzuli

Jezyk Definiowania Danych; DDL

W sklad tego jezyka wchodza instrukcje:

  • CREATE - Tworzy nowy obiekt
  • ALTER - Modyfikuje obiekt
  • DROP - usuwa obiekt

W zaleznosci jaki efekt chcemy uzyskac, te polecenia moga byc mniej lub bardziej rozbudowane.

TWORZENIE BAZY DANYCH to polecenie:
CREATE DATABASE [nazwa bazy];

ATRYBUT OPIS
Primary key - wskazanie kolumny klucza glownego
NOT NULL - nie pozwala na pozostawianie pustego pola
AUTO_INCREMENT - utomatyczne numerowanie
DEFAULT - wartosc domyslna
UNIQUE - nie pozwala na powtarzanie wartosci
CHECK - tworzy warunek ograniczajacy wprowadzane wartosci

DML - Jezyk manipulowania danymi

DML sklada sie z czterech (4) polecen:

  • INSERT - Wstawia dane do bazy danych.
  • SELECT - Odczytuje dane z bazy.
  • UPDATE - Modyfikuje przypisana wczesniej wartosc
  • DELETE - Usuwa rekordy
